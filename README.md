# Agentic AI Business Advisor and Compliance System 🚀

An intelligent, AI-driven platform built on the **MERN Stack** (MongoDB, Express, React, Node.js) and powered by **Google Gemini 2.5 Flash**. This application acts as a virtual business consultant, expertly guiding aspiring entrepreneurs in India to launch robust, budget-aligned businesses while strictly navigating state-specific legal compliances.

## 🌟 Key Features

- **Geographic Demographic Analysis:** Input any Indian city or area, and the AI will analyze local market trends to suggest the 5 highest-potential business ideas for that specific demographic.
- **Custom Idea Validation:** Already have a business idea? Skip the suggestions and directly input your custom concept for AI evaluation.
- **Dynamic Budget Allocation:** Input your available capital (e.g., 5 Lakhs), and the system generates a meticulously mapped financial roadmap allocating funds to rent, inventory, marketing, and operations.
- **Strict State Compliance Checking:** Automatically generate comprehensive legal checklists tailored to your business idea and location—covering FSSAI, Trade Licenses, GST, Fire Safety NOCs, and local municipal laws.
- **Premium UI/UX:** A stunning, modern interface built with React and Vanilla CSS featuring glassmorphism elements, dark mode aesthetics, and micro-animations.

## 🛠️ Tech Stack

- **Frontend:** React.js (Vite), React Router, Lucide Icons, React Markdown (Vanilla CSS).
- **Backend:** Node.js, Express.js.
- **Database:** MongoDB (via Mongoose).
- **AI Integration:** Google Generative AI (`@google/generative-ai`) utilizing the blazing-fast `gemini-2.5-flash` model.

## ⚙️ How to Run Locally

### Prerequisites
- [Node.js](https://nodejs.org/) (v20.17+ recommended)
- [MongoDB](https://www.mongodb.com/) (running locally on default port 27017)
- A Google Gemini API Key from [Google AI Studio](https://aistudio.google.com/app/apikey)

### 1. Backend Setup
1. Open a terminal and navigate to the `backend` folder:
   ```bash
   cd backend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set up your environment variables:
   - Open `.env` (or create one using `.env.example`).
   - Add your Gemini API Key:
     ```env
     PORT=5000
     GEMINI_API_KEY=your_actual_api_key_here
     ```
4. Start the backend server:
   ```bash
   node server.js
   ```

### 2. Frontend Setup
1. Open a new terminal and navigate to the `frontend` folder:
   ```bash
   cd frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the Vite React development server:
   ```bash
   npm run dev
   ```

Navigate to `http://localhost:5173` in your browser to meet your new AI Business Advisor!

## 🔐 Security Note
This project uses a `.gitignore` file to ensure that `.env` files (which contain sensitive API keys) are not uploaded or exposed to GitHub. Please **never commit your literal API keys** to public repositories.
